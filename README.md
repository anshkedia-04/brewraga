## 📌 Overview
**BrewRaga** is a modern, single-page website designed for a high-end rooftop café in Vadodara. It provides a seamless user experience for customers to explore the menu, view daily specials, and book tables.

The site features a **custom-built Admin Dashboard** (Client-side) that allows staff to update menu items and specials dynamically without touching the code.

---

## ✨ Features
✅ **Dynamic Menu System** – Tabbed navigation for Hot/Cold drinks and Food.  
✅ **WhatsApp Reservation** – Direct integration to book tables via WhatsApp.  
✅ **Staff Admin Mode** – Built-in CMS to add/edit/delete menu items (Password protected).  
✅ **Interactive UI** – Glassmorphism effects, scroll-spy navigation, and reveal animations.  
✅ **Chatbot UI** – A simulated AI assistant for customer queries.  
✅ **Local Storage Persistence** – Menu updates are saved locally for the device.  

---

## 🧠 How It Works
1. **User Experience:** Visitors can browse the "Today's Special" (updated daily), filter the menu, and use the "Reserve a Table" form which formats their request and opens WhatsApp.
2. **Staff Mode:** 
   - Click **"Staff Login"** in the footer.
   - Enter Password: `admin123`.
   - **Edit Mode** activates: Click any menu item to edit text/price, or use the delete buttons.
   - **Add Items:** Use the "+ Add Special" or "+ Add Item" buttons to expand the menu.
   - Changes persist via the browser's `localStorage`.
